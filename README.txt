SETUP INSTRUCTIONS:
-------------------
1. Create a folder named "maxytech-crt" on your computer

2. Inside this folder, create the following files:
   - index.html
   - portfolio.html  
   - contact.html
   - style.css
   - script.js

3. Create an "images" folder inside maxytech-crt

4. Add your images to the images folder:
   - logo.png (your company logo)
   - profile.jpg (your profile photo)
   - graphic-design1.jpg, graphic-design2.jpg
   - music1.jpg, music2.jpg
   - website1.jpg, website2.jpg
   - video1.jpg, video2.jpg
   - social-media.jpg

5. Update WhatsApp number:
   - In script.js, replace '1234567890' with your actual WhatsApp number
   - Also update in the WhatsApp float button links in all HTML files

6. Update contact information:
   - In contact.html, update the phone number, email, and address

7. Open index.html in your browser to view the website

CUSTOMIZATION:
--------------
- Colors: Edit the CSS variables in style.css (--primary-blue, etc.)
- Content: Update text in HTML files
- Images: Replace with your own project images
- Social Links: Update URLs in footer and contact page
